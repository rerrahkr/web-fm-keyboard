# FM Keyboard

A web synthesizer for YM2608.

A tiny experimental project to use [ymfm](https://github.com/aaronsgiles/ymfm) and [blip_buf-rs](https://github.com/nomyfan/blip_buf-rs) in AudioWorklet through WASM worker.
